Funciones del kernel y su interacción con el hardware:

- Gestión de memoria: asigna y libera memoria para procesos y datos.
- Gestión del procesador: asigna tiempo de CPU a procesos y gestiona la ejecución de tareas.
- Gestión de dispositivos: controla la comunicación entre software y hardware mediante controladores.
- Gestión de archivos: provee interfaces para manipulación de archivos y directorios.
- Seguridad: implementa mecanismos para proteger el sistema y los datos.

El kernel interactúa con el hardware a través de los controladores de dispositivo. Cuando un proceso necesita acceder a un dispositivo, el kernel lo gestiona mediante el controlador correspondiente, actuando como intermediario.

Gestión de procesos y multitarea:

- Los procesos son instancias en ejecución de un programa. El sistema operativo gestiona los procesos, asignándoles recursos y tiempo de CPU.
- La multitarea permite que múltiples procesos se ejecuten simultáneamente, compartiendo el tiempo de CPU. Mejora la eficiencia del sistema al mantener ocupada la CPU y permitir la ejecución de múltiples tareas concurrentemente.

Medidas de seguridad a nivel de sistema operativo para proteger una red:

- Cortafuegos (Firewall): controla el tráfico de red entrante y saliente, permitiendo o bloqueando según las reglas configuradas.
- Listas de Control de Acceso (ACL): establece permisos de acceso a recursos de red basados en direcciones IP, puertos, protocolos, etc.
- Actualizaciones de seguridad: mantener el sistema operativo y el software actualizados para parchear vulnerabilidades conocidas.
- Autenticación y Autorización: requerir credenciales para acceder a recursos de red y limitar los privilegios de usuario.
- Detección de Intrusos: monitorea y alerta sobre actividades sospechosas en la red.
- Encriptación: protege la comunicación entre sistemas mediante el cifrado de datos transmitidos.

Estas medidas ayudan a proteger la red empresarial contra amenazas externas e internas, garantizando la confidencialidad, integridad y disponibilidad de los datos y recursos.
